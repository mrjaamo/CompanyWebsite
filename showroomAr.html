Show Room AR

Showroom AR is meant for event organizers to create augmented reality events that the users can access and view. They can open events from a list that is sorted by: cities, the events, and their names inside the city. Events can currently showcase art samples and will be able to show videos and 3D objects in the future.

Core App Mechanics

On the first launch of the app, the user is directed to complete a tutorial that is basically an offline mode of the event. This will ask the user to provide permissions for the gyroscope and camera of the phone, which are required for the AR Core to work. This tutorial is primarily meant to test if the app is even compatible with the user's phone. After this, the user can proceed to browsing the event list and choose an event.

On the event, the user can tap on the pictures, view information about them, and like them.

New Updates

Now Users can preview different art on their own walls. With this, users could test if their desired art element would work in their own room or enviroment
